# Operation-Quasar-Fire
## Requeriments
- Java 17
- Maven 3.9.1

## Compile and deploy
In the root of the project run the next commands:
### Windows or Linux:
```bash
mvn clean install
java -jar target/firequasar-0.0.1-SNAPSHOT.jar
```
## Remember!
Make sure you have java version 17 and the JAVA_HOME environment variable configured!
